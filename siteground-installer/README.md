# CARBON[6] One-Time Install Code System (SiteGround)

**Simple PHP system for secure installer distribution**

---

## 🎯 WHAT IS THIS?

A one-time use code generator for your CARBON[6] Council System installer.

**You generate unique codes** → **Share with users** → **Each code works once**

---

## ⚡ QUICK START

1. **Upload 4 files** to `/public_html/install/` via cPanel
2. **Edit config.php** - set your admin password
3. **Visit** thecarbon6.agency/install
4. **Generate codes** and share

**Full setup guide:** See SETUP.md

---

## 📁 FILES

- `config.php` - Configuration (set password here!)
- `index.php` - Admin dashboard
- `redirect.php` - Code validator
- `.htaccess` - URL routing
- `SETUP.md` - Complete setup instructions

---

## 🔒 SECURITY

✅ One-time use enforcement
✅ Auto-expiration (1hr - 30 days)
✅ Password protected admin
✅ SQLite database (auto-created)
✅ Secure file permissions
