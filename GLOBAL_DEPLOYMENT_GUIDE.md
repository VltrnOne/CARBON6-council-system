## 🆘 TROUBLESHOOTING GUIDE

### Problem: "Python not found"
```bash
# Solution: Install Python 3.9+
# macOS:
brew install python@3.11

# Ubuntu/Debian:
sudo apt install python3.11

# Windows:
# Download from python.org
```

### Problem: "Permission denied"
```bash
# Linux/macOS:
chmod +x install_sovereign_manifold*.sh

# Windows:
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### Problem: "Out of memory during install"
```bash
# Solution: Increase swap or use cloud instance
# Temporary fix (Linux):
sudo fallocate -l 4G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
```

### Problem: "Tests fail after install"
```bash
# Check Python version
python --version  # Should be 3.9+

# Reinstall packages
pip install --force-reinstall -r requirements-manifold.txt

# Verify NumPy works
python -c "import numpy; print(numpy.__version__)"
```
